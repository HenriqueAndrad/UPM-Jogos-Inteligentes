Para utilizar as operações de seleção, ordenação, e mutação que foram escritas neste código, o usuário deve analisar qual das opções é a mais otimizada para se encaixar no seu projeto. 

Após essa análise, o primeiro passo que o usuário deve ter é implementar a classe genérica “Genome<T>” assim como foi implementada neste código. Essa classe será responsável por criar e definir valores que serão usados em outras operações, portanto é essencial para o funcionamento do código.

Depois disso o usuário deverá implementar outra classe genérica de métodos onde ele definirá variáveis globais que serão usados em grande parte das operações, o valor dessas variáveis deve ser ajustada para que se adeque ao projeto.

Quando todos os valores globais tiverem sido definidos, basta o usuário copiar qual operação ele deseja e, caso necessário, deve definir os valores das variáveis locais da função específica escolhida.
